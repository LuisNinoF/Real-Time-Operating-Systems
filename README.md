# Real-Time-Operating-Systems

## Objective
Show the use of real time operating systems (RTOS) and FreeRTOS. There are two projects: 

P1 RTOS - shows a custom round robin scheduler with context switching between three threads using a linked list structure.

P2 FreeRTOS - shows a custom application using FreeRTOS, including: interrupts, notifications, queues, semaphores and mutex.

### Skills Learned

- Round robin scheduling
- Context switching
- RTOS
- FreeRTOS

### Tools Used

- STM32CubeIDE

### Languages

- Assembly
- C
