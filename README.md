# Real-Time-Operating-Systems

## Objective
Show the use of real time operating systems (RTOS) and FreeRTOS. There are two projects: 

P1 RTOS - shows my own scheduler for context switching between three threads using a linked list structure.

P2 FreeRTOS - shows an application using FreeRTOS, including: interrupts, notifications, queues, semaphores and mutex.

### Skills Learned

- Context switching
- RTOS
- FreeRTOS

### Tools Used

- STM32CubeIDE

### Languages

- Assembly
- C
